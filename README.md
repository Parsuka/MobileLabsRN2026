
# MobileLabsRN2026

## Лабораторна робота №1: Найпростіший додаток React Native

### Опис проєкту
Створення мобільного додатка на базі Expo з трьома екранами: Головна (Новини), Фотогалерея та Профіль (Реєстрація).

### Інструкція із запуску
1. Перейдіть у папку з проєктом: `cd lab1`
2. Встановіть залежності: `npm install`
3. Запустіть проєкт: `npx expo start --tunnel`

### Способи запуску та тестування
* **Expo Go (Physical Device):** Найкращий спосіб для перевірки реальної продуктивності та жестів.
* **Android Emulator:** Зручно для розробки без фізичного пристрою.
* **Web Browser:** Швидка перевірка верстки, але не підтримує всі нативні функції.
